To Write Java Script Code We Can Use ---Script--- Tag In HTML Script Tag Can Be Used In Head As Well As Body But It's Better To Write Script Tag Before The Closing Body Tag Because Script Interption Can Slow Down The Website So Use It Before Closing Body Tag.

There Are Two Ways To Write Java Script Code

1] Internal Java Script --- Inside Script Tag
2] Exnternal Java Script File --- Linking To HTML With src Attribute By Giving Proper Path Of External JS File.

If You Will Do Both The src File Will Be Excuted And The Code Inside Script Tag Will Be Ignored.

All The Code Output Is In Console Part Of Developer Tools.
